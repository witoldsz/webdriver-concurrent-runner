Libraries
=========

1. Put http://selenium.googlecode.com/files/selenium-server-standalone-2.13.0.jar
into lib folder.
2. Download http://google-guice.googlecode.com/files/guice-3.0.zip unpack somewhere and copy:

    *   guice-3.0
    *   aopalliance.jar
    *   javax.inject

    into lib folder.
